### 如何使用 TinaCMS 编辑内容

现在您可以通过以下步骤访问和使用 TinaCMS 编辑功能：

1. 访问编辑模式 ：

   - 访问 http://localhost:3000/admin 页面
   - 点击「内容预览」按钮
   - 在打开的预览页面 URL 末尾添加 ?edit 参数（例如： http://localhost:3000/api/tina/preview?edit=true ）
   - 或者直接访问 http://localhost:3000?edit=true 来编辑首页

2. 使用编辑功能 ：

   - 进入编辑模式后，右下角会出现一个「编辑内容」按钮
   - 点击此按钮可以激活 TinaCMS 编辑器
   - 您可以直接在页面上编辑文本内容，修改后系统会自动保存到 content/home/index.mdx 文件中

3. 查看更改 ：

   - 编辑完成后，您可以刷新页面查看更新后的内容
   - 所有更改都会直接反映在 Markdown 文件中

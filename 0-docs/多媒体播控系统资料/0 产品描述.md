# 智能展馆多媒体中控系统

## 产品概述

智能展馆多媒体中控系统是一款基于ESP32技术的高性能、高可靠的展馆设备集中控制解决方案。该系统具有以下核心特点：

1. **精细设备控制**：通过控制面板实现对投影机、电脑主机开关机、照明灯电源、播放器、播放软件和音视频内容的精细控制
2. **独立群组运行**：每个子展项可内部独立成组，无需依赖外网和服务器，实现设备间的本地链接和通讯
3. **灵活群组集成**：当需要集成多个群组（展项）时，可快速接入中央服务器，并通过平板电脑进行集中控制，支持多级权限管理
4. **高性能播放终端**：采用基于Linux的嵌入式播放器，内置画面几何校正和H5页面渲染功能，充分利用GPU和解码器硬件加速能力

该系统通过群组化设计和分布式架构，为观众提供沉浸式、无缝的展览体验，为展馆运营方提供高效、便捷的管理工具。

## 系统架构

### 多层级分布式设计

系统采用创新的群组化架构，实现了"独立运行与集中管理"的双重优势：

1. **独立群组结构**：每个子展项作为独立群组，由ESP32Touch面板作为主节点，群组内部通过BLE GATT实现设备间本地通信，无需依赖外网和服务器
2. **群组快速集成**：支持群组的即插即用式接入，通过WiFi连接到中央MQTT服务器，实现跨群组的统一控制
3. **灵活扩展**：可根据展览规模动态调整群组数量和组织结构，支持从单一展项到大型展览馆的全方位覆盖

### 核心组件

- **中央控制系统**：MQTT服务器作为群组集成核心，支持多展项集中管理和多级权限控制
- **群组主节点**：ESP32Touch面板，作为每个子展项的控制中心和人机交互界面，实现对所有设备的精细化控制
- **设备控制节点**：ESP32继电器模块（控制电源开关）、ESP32 RS232模块（控制串口设备）、ESP32 HID模块（模拟键盘鼠标）等
- **智能播放终端**：基于Linux的嵌入式播放器，内置GPU硬件加速和硬件解码器，支持画面几何校正和H5页面渲染
- **移动控制终端**：平板电脑APP，提供直观的集中控制界面和权限管理功能

## 主要功能

### 智能设备控制

- **全设备精细控制**：实现对投影仪（开关、信号切换、亮度调节）、电脑主机（开关机、休眠唤醒）、照明灯电源（开关、亮度、色温）、播放器（开关、音量调节）、播放软件（启动关闭、模式切换）和音视频内容（播放、暂停、切换）的精细化控制
- **多协议统一控制**：通过RS232串口、USB HID模拟键盘鼠标、蓝牙HID、继电器等多种方式实现设备的统一管理
- **一键场景切换**：预设多种展览场景，一键触发相关设备的联动控制

### 高性能播放与渲染

- **嵌入式Linux播放器**：基于Linux系统的专用播放终端，确保系统稳定运行
- **硬件加速渲染**：充分利用GPU硬件加速能力，支持复杂H5页面和WebGL内容的流畅展示
- **专业画面处理**：内置画面几何校正功能，解决不规则投影面的图像失真问题
- **硬件解码器**：配备专用视频解码器，支持4K/8K高清内容的流畅播放

### 群组管理与权限控制

- **独立群组运行**：每个子展项可在无外网、无服务器的情况下独立运行，确保基础功能正常
- **集中控制平台**：通过平板电脑APP实现对多个群组（展项）的集中管理
- **多级权限管理**：支持设置不同级别的操作权限，确保系统安全和管理规范
- **快速群组集成**：支持新群组的快速添加和配置，无需复杂的网络设置

## 技术特色

### 高性能播放与渲染

- **嵌入式Linux架构**：采用稳定可靠的Linux操作系统，确保长时间稳定运行
- **GPU硬件加速**：充分利用硬件GPU进行图形渲染，提升H5和WebGL内容的展示效果
- **专用解码器**：配备硬件视频解码器，支持多种编码格式，确保高清内容的流畅播放
- **几何校正技术**：内置专业的画面几何校正算法，解决异形投影面的图像问题

### 独立运行与灵活集成

- **本地通信架构**：群组内部通过BLE GATT实现设备间的本地通信，无需依赖外部网络
- **自组织网络**：设备能够自动发现并连接到群组，简化安装和配置流程
- **即插即用扩展**：支持设备的热插拔和自动识别，便于系统维护和升级
- **快速集成中央系统**：需要集中控制时，可迅速接入中央服务器，实现跨群组管理

### 精细控制与权限管理

- **多协议兼容**：同时支持RS232串口、USB HID、蓝牙HID、继电器等多种控制协议
- **设备状态实时监控**：实时监测各设备的运行状态，提供故障预警和报警功能
- **多级权限控制**：基于角色的访问控制，支持管理员、操作员等不同级别的权限设置
- **操作日志记录**：详细记录所有操作，便于系统审计和问题追溯

## 系统规格

### 硬件规格

- **主控设备**：
  - ESP32双核处理器，支持BLE 5.0和WiFi 4/5
  - 触摸控制面板，提供直观的操作界面
  
- **控制节点**：
  - ESP32继电器模块：控制电源开关和通断
  - ESP32 RS232模块：支持标准串口设备通信
  - ESP32 HID模块：模拟键盘鼠标操作
  
- **播放终端**：
  - 基于高性能SoC的Linux嵌入式播放器
  - 内置GPU加速和硬件视频解码器
  - 支持4K/8K内容播放
  - 存储容量不低于128GB
  - 支持多HDMI输出，每接口支持4K分辨率
  
- **接口规格**：
  - 控制接口：RS232、USB、蓝牙、WiFi
  - 音频输出：3.5mm音频接口
  - 电源要求：支持220V交流输入

### 软件规格

- **操作系统**：Linux嵌入式系统
- **核心功能**：
  - 内置画面几何校正算法
  - 支持复杂H5页面渲染
  - 硬件加速的媒体播放
  
- **通信协议**：
  - 本地通信：BLE GATT
  - 网络通信：MQTT、WiFi
  
- **权限管理**：基于角色的多级权限控制
  
- **移动控制**：支持平板电脑APP集中控制

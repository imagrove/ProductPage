# 智能展馆多媒体中控系统

## 产品概述

智能展馆多媒体中控系统是一款基于ESP32技术的高性能、高可靠的展馆设备集中控制解决方案。该系统通过群组化设计和分布式架构，实现了对展馆内多种多媒体设备的智能控制、内容管理和远程运维，为观众提供沉浸式、无缝的展览体验，为展馆运营方提供高效、便捷的管理工具。

## 系统架构

### 多层级分布式设计

系统采用创新的群组化架构，将每个子展项作为独立群组，由ESP32Touch面板作为主节点进行控制。群组内部各设备通过BLE GATT进行高效通信，群组间则通过WiFi连接到中央MQTT服务器，实现了集中管理与分布式控制的完美结合。

### 核心组件

- **中央控制系统**：MQTT服务器作为系统核心，负责各群组间的通信协调和集中控制
- **群组主节点**：ESP32Touch面板，作为每个子展项的控制中心和人机交互界面
- **设备控制节点**：ESP32继电器、ESP32 RS232、ESP32 HID和ESP32传感器等多种功能模块
- **播放终端**：基于Linux嵌入式的Web播放系统，支持高清内容展示和多设备协同
- **OTA服务平台**：提供设备固件的远程升级和管理，确保系统持续优化和安全

## 主要功能

### 智能设备控制

- **多类型设备支持**：投影仪、音频设备、灯光设备、计算机主机等多种设备的集中控制
- **多样化控制方式**：支持RS232串口控制、USB HID控制、蓝牙HID控制和继电器控制
- **环境感知**：通过传感器实时采集环境数据，支持智能化场景切换

### 多媒体内容展示

- **高清内容播放**：支持HTML5视频/音频/图片的流畅播放，具备硬件加速能力
- **交互式内容呈现**：支持基于JavaScript、CSS3、WebGL开发的交互式体验内容
- **几何校正与边缘融合**：通过Three.js实现精准的几何校正和多屏无缝拼接
- **分布式画面融合**：支持超大分辨率画面分配给多个终端协同展示，最大支持16K×8K

### 系统管理与运维

- **集中管理平台**：管理员通过终端远程监控和控制整个系统
- **OTA远程升级**：基于MQTT推送和HTTPS安全通道的设备固件远程升级服务
- **容器化部署**：采用Docker Compose实现环境隔离和快速部署
- **全链路监控**：详细的设备状态监控、操作日志记录和系统运行分析

## 技术特色

### 高性能图形渲染

- **WebGL硬件加速**：利用GPU实现复杂视觉效果的高效渲染
- **Three.js 3D图形库**：提供丰富的3D内容开发能力
- **硬件视频解码**：支持H.264/HEVC/VP8/VP9/AV1等多种编码格式，确保4K/8K内容流畅播放

### 安全与可靠性

- **端到端安全**：HTTPS加密传输、固件校验机制，保障系统安全
- **系统防护**：Linux安全机制、应用沙箱环境、看门狗机制多重保障
- **异常恢复**：系统异常时自动重启或恢复到正常状态，确保稳定运行

### 灵活扩展能力

- **模块化设计**：各功能模块独立封装，方便扩展和替换
- **丰富接口**：支持RS232、USB、蓝牙、WiFi等多种通信方式
- **POE供电支持**：可选配POE供电模组，简化布线和安装

## 系统规格

### 硬件规格

- **主控制器**：ESP32双核处理器，支持BLE 5.0和WiFi 4/5
- **存储容量**：播放终端建议存储容量不低于128GB
- **显示输出**：支持多HDMI输出，每接口支持4K分辨率
- **音频接口**：3.5mm音频输出
- **电源要求**：支持220V交流输入，可选POE供电

### 软件规格

- **操作系统**：Linux嵌入式系统
- **浏览器引擎**：Chromium/WebKit定制版本
- **图形库**：WebGL、Three.js
- **网络协议**：MQTT、HTTPS、WebSocket
- **安全加密**：TLS/SSL，固件数字签名

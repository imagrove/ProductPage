# TinaCMS Product Management Guide

## Accessing the Admin Panel
1. Navigate to: http://localhost:3000/admin/index.html
2. You'll see the TinaCMS interface with your products

## Adding a New Product
1. Click "Products" in the left sidebar
2. Click "Create New" button
3. Fill in the product details:
   - **Product Name**: Enter your product title
   - **Description**: Write a compelling product description
   - **Price**: Set the product price (e.g., 199.99)
   - **Product Image**: Upload an image or enter image URL
   - **SKU**: Create a unique SKU (e.g., PROD-001)
   - **Stock Quantity**: Set available inventory

## Editing Existing Products
1. Click on any product in the list
2. Edit the fields as needed
3. Changes are saved automatically

## Product Content Tips
- Use clear, descriptive titles
- Write detailed descriptions with features and benefits
- Use high-quality product images
- Set competitive prices
- Keep SKU format consistent
- Monitor stock levels

## Image Management
- Upload product images directly through TinaCMS
- Recommended image size: 800x600px or larger
- Use consistent image dimensions for better display
- Consider using product lifestyle images

## Content Best Practices
- **SEO-Friendly Titles**: Include key product features
- **Detailed Descriptions**: Cover specifications, benefits, use cases
- **Pricing Strategy**: Research competitor pricing
- **Inventory Management**: Keep stock quantities updated
- **Product Categories**: Consider organizing by product type

## Preview and Publishing
- Changes are visible immediately in development
- Test product pages at: http://localhost:3000/products/[product-slug]
- Verify cart functionality and payment integration
